Probable Cause:

The error message indicates that there is a potential object cycle in the data structure being serialized to JSON.
This can occur when an object references itself directly or indirectly, creating a loop. 
In this case, it seems that the `Student` object has a property `ExamResults`, which in turn references the `Student` 
object again, leading to a cycle.

Solution:

Add to program.cs:

//To avoid the circular reference problem when serializing objects to JSON
//Solution for Error :A possible object cycle was detected ...
builder.Services.AddControllers().AddNewtonsoftJson(options =>
   options.SerializerSettings.ReferenceLoopHandling = Newtonsoft.Json.ReferenceLoopHandling.Ignore
);
