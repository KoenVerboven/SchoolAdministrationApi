  public async Task<ActionResult<Student>>GetStudentById(int id)
        {
            var student = await _studentRepository.GetByIdAsync(id);

            if (student == null)
            {
                return NotFound();
            }

            return Ok(student);
        }

 --> async Task -> the task is executed asynchronously.
 Bij een asynchrone task hoort ook steeds het keyword 'await'.
 Await is used for an action that will last a certain amount of time.
 For example, retrieving a result from the database : await _studentRepository.GetByIdAsync(id);

 The Task here is of the generic type <ActionResult<Student>>

 We must therefore return an actionResult of the student type.

 an actionResult can be, for example : 
 Ok (statusCode 200), 
 CreatedAtAction (statusCode 201), 
 NoContent (statuscode 204),
 BadRequest(statuscode 404),.......
